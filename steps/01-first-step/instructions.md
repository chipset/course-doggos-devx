# Intro

You are in the DOGGOS course.  During this course, you will learn how to navigate screens.  You will update code, build it, then debug it all without logging into a 3270 session.  

You are in a secure cloud environment which runs VSCode and is connected to the Mainframe.  

Get familiar with the Activity Bar ![Activity Bar](assets/activityBar.png)
## Your task

Make sure the initial build process has been completed successfully (**exit code: 0** message in the active terminal)

Once done, click **Check Work** to continue.
